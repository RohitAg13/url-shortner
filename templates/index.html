ushrink

This project was done with the main objective being to learn and practice the application of some tools used here, like Flask and Bootstrap, as well as pure CSS. I will try to maintain this URL shortener up and running, so it can be used as meant, but I can't guarantee stability in high intensity usage, once this will be hosted at a free Heroku dino.

Hope you enjoy.

This project runs under Flask, a Python microframework. To learn more, visit:

Flask Official Website

As front-end framework, we used Bootstrap 4 alpha. To learn more, visit:

Bootstrap 4-alpha Docs
